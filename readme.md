# 🔗 URL Shortener with Authentication

A full-stack **URL Shortener Web Application** with user authentication, built using **Flask (backend)** and **React (frontend)**.  
Users can sign up, log in, shorten URLs, and manage their own links.

---

## 🚀 Features

- 🔐 User Signup & Login system
- 🔑 Password hashing using bcrypt
- 🔗 Create short URLs from long URLs
- 👤 User-specific dashboard
- 🔁 Redirect short URLs to original links
- 📋 Copy short URL to clipboard
- 🚪 Logout functionality
- 🔒 Protected frontend routes
- 🧩 Modular backend architecture

---

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router DOM
- Axios
- JavaScript (ES6)

### Backend
- Flask (Python)
- Flask-CORS
- MySQL
- bcrypt (password hashing)
